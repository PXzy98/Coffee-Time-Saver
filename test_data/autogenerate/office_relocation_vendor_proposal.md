## 1. Background

## 3. IT Infrastructure Relocation

The IT team (lead: Marcus Webb) has assessed the following requirements for the new premises:
- Install 240 data drops (Cat6A) across Floors 12–14
- Relocate 3 server racks from current data room to new server room on Floor 12
- Provision fibre uplink from street (ISP: Rogers Business)
- Install new Wi-Fi access points (48 units, Cisco Meraki)

**Current status:** Rogers Business has confirmed fibre availability at 1200 Bay St but requires a **signed service agreement by April 15, 2026** to guarantee the July installation date. If the deadline is missed, next available slot is October 2026.

**Estimated IT relocation cost:** $127,000 (hardware) + $34,000 (labour)
